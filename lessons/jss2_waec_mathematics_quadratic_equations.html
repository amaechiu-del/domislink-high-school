Quadratic Equations

A quadratic equation is a polynomial equation of degree two, which means the highest power of the variable is two. It has the general form of ax^2 + bx + c = 0, where a, b, and c are constants.

Here are the key points to note about quadratic equations:

  • The coefficient of the x^2 term (a) cannot be zero.
  • The equation can be solved using factoring, the quadratic formula, or graphing.
  • The solutions to the equation are called roots or zeros.

Let's consider two examples from Nigeria:

Example 1: A farmer in Kano State has 100 meters of fencing to enclose a rectangular plot of land. If the length of the plot is x meters, find the width in terms of x.

Solution: Let's assume the width is y meters. The perimeter of the plot is 2x + 2y, which is equal to 100 meters. So, we have the equation 2x + 2y = 100, which can be simplified to x + y = 50, and then y = 50 - x.

Example 2: A shop owner in Lagos State sells a certain product for ₦200 per unit. The cost of producing x units is ₦100x + ₦500. If the revenue from selling x units is ₦200x, find the profit in terms of x.

Solution: The profit is the difference between the revenue and the cost, so we have the equation Profit = Revenue - Cost, which is ₦200x - (₦100x + ₦500), and then ₦100x - ₦500.

Fun Fact: Did you know that quadratic equations have many real-world applications, such as predicting the trajectory of a projectile, modeling population growth, and optimizing business processes?

Now, let's practice solving some quadratic equations:

Question 1: Solve the equation x^2 + 4x + 4 = 0

Answer: (x + 2)(x + 2) = 0, so x = -2

Question 2: Solve the equation x^2 - 7x + 12 = 0

Answer: (x - 3)(x - 4) = 0, so x = 3 or x = 4

Question 3: Solve the equation x^2 + 2x - 6 = 0

Answer: Using the quadratic formula, x = (-b ± √(b^2 - 4ac)) / 2a, where a = 1, b = 2, and c = -6, we get x = (-2 ± √(2^2 - 4(1)(-6))) / 2(1), so x = (-2 ± √(4 + 24)) / 2, and then x = (-2 ± √28) / 2, and finally x = (-2 ± 2√7) / 2, which simplifies to x = -1 ± √7
