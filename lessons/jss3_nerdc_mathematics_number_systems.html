Number Systems

In mathematics, a number system is a set of numbers and operations used to represent and manipulate quantities. The most common number systems are the decimal system, binary system, and hexadecimal system.

The following are explanations of number systems in bullet points:

  • Decimal system: a number system with base 10, using the digits 0-9.
  • Binary system: a number system with base 2, using the digits 0 and 1.
  • Hexadecimal system: a number system with base 16, using the digits 0-9 and letters A-F.

Fun Fact

The first computer bug was an actual insect that got stuck in the Harvard Mark II computer in 1947. The operators were trying to debug the system when they found the moth, and from then on, any glitch or error was called a "bug".
