Logarithms — Key Concepts

  • Definition: If ax = N, then x = loga N (a > 0, a ≠ 1).
  • Base 10: Common logarithms — written as log N (no base shown).
  • Base e: Natural logarithms — written as ln N.
  • Laws of Logarithms:
    • log (AB) = log A + log B
    • log (A/B) = log A – log B
    • log (An) = n log A
  • Change of Base: loga b = log b / log a

🇳🇬 Nigerian Examples

Example 1 (WAEC-style): If log10 2 = 0.3010 and log10 3 = 0.4771, find log10 12.

Solution: 12 = 2² × 3, so log 12 = 2 log 2 + log 3 = 2(0.3010) + 0.4771 = 0.6020 + 0.4771 = 1.0791.

Example 2 (Nigerian Market Problem): A trader's profit (₦) grows such that P = 500 × 2t, where t is years. How long to reach ₦4000?

Solution: 4000 = 500 × 2t → 8 = 2t → t = log₂ 8 = 3. 3 years.

💡 Fun Fact

The word "logarithm" comes from Greek logos (ratio) and arithmos (number). John Napier invented them in 1614 — they were once used to multiply huge numbers by just adding their logs! In Nigeria, log tables were essential for JAMB and WAEC exams before calculators became common.

Practice Questions

  1. Evaluate: log5 125 + log2 16
  2. If log10 7 = 0.8451, find log10 49.
  3. Solve for x: log3 (x + 1) = 2

✅ Answers

  1. log5 125 = 3 (since 5³=125), log2 16 = 4 (2⁴=16) → 7
  2. log 49 = log 7² = 2 × 0.8451 = 1.6902
  3. x + 1 = 3² = 9 → x = 8
