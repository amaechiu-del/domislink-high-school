Integral Calculus (SSS3 WAEC Mathematics)

Key Concepts

  • Integration is the reverse process of differentiation (antidifferentiation).
  • Definite Integral ∫ab f(x) dx gives the area under the curve y = f(x) from x = a to x = b.
  • Indefinite Integral ∫ f(x) dx = F(x) + C, where C is the constant of integration.
  • Power Rule: ∫ xn dx = (xn+1)/(n+1) + C, for n ≠ -1.
  • The Fundamental Theorem of Calculus links differentiation and integration.

Nigerian Examples

Example 1 (Lagos Market Revenue):
A trader's marginal revenue (in ₦) is given by MR(x) = 2x + 5, where x is the number of bags of rice sold. Find the total revenue function R(x) and the revenue from selling 10 bags.

Solution:
R(x) = ∫ (2x + 5) dx = x² + 5x + C. When x=0, R(0)=0 → C=0.
So R(x) = x² + 5x. Revenue for 10 bags: R(10) = 100 + 50 = ₦150.

Example 2 (Kano Irrigation Canal):
The cross-section of a canal is modeled by y = 4 - x² from x = -2 to x = 2 (in meters). Find the area of the cross-section.

Solution:
Area = ∫-22 (4 - x²) dx = [4x - x³/3]-22
= (8 - 8/3) - (-8 + 8/3) = (16/3) - (-16/3) = 32/3 = 10.67 m².

🎯 Fun Fact: Integration was independently developed by Isaac Newton and Gottfried Leibniz in the 17th century. The integral sign ∫ is actually an elongated letter 'S', standing for "summa" (Latin for "sum")!

Practice Questions

  1. Question 1: Evaluate ∫ (3x² - 2x + 1) dx.
  2. Question 2: Find the area bounded by the curve y = x² + 1, the x-axis, and the lines x = 0 and x = 3.
  3. Question 3: If dy/dx = 4x³ - 6x and y = 5 when x = 1, find y as a function of x.

Answers

  1. Answer 1: x³ - x² + x + C
  2. Answer 2: ∫03 (x²+1) dx = [x³/3 + x]03 = (9 + 3) - 0 = 12 square units
  3. Answer 3: y = ∫ (4x³ - 6x) dx = x⁴ - 3x² + C. Using y=5 when x=1: 5 = 1 - 3 + C → C = 7. So y = x⁴ - 3x² + 7
